# 电商订单管理系统

## 项目简介
这是一个适用于小微企业的电商订单管理系统，支持订单创建、支付处理、库存管理等核心功能。系统采用Spring Boot框架开发，具有高性能和可扩展性。

## 主要功能
1. 订单管理
   - 订单创建：支持商品选择和数量确认
   - 订单状态管理：未支付、已支付、已发货、已完成、已取消
   - 订单查询：支持按订单号和用户ID查询

2. 支付集成
   - 支持支付宝支付
   - 支付状态实时回调
   - 支付结果自动更新

3. 库存管理
   - 下单时自动锁定库存
   - 支付超时自动释放库存
   - 库存不足时订单创建失败

## 技术栈
- 后端框架：Spring Boot 2.x
- 数据库：MySQL 5.7+
- ORM框架：JPA
- 项目管理：Maven
- 支付集成：支付宝SDK

## 系统要求
- JDK 1.8+
- Maven 3.6+
- MySQL 5.7+
